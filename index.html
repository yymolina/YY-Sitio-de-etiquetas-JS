script

Permite la inserción de códigos script, como puede ser JavaScript en un documento web del lado del cliente. El código se puede insertar directamente entre las etiquetas o haciendo referencia a un documento externo el cual contiene el código script a ejecutar. Ejemplo:

<script src="js/ejemplo.js"></script>

link

Es un elemento clave en el desarrollo y diseño web, ya que permite establecer relaciones y conexiones entre el documento HTML y recursos externos, como hojas de estilo CSS, íconos de favoritos, fuentes web, entre otros. Ejemplo:

<link rel="ejemplo" href="www.ejemplo.com">

meta

Define metadatos sobre el documento, como el juego de caracteres, la descripción y palabras clave. Ejemplo:

<meta name="nombre" content="yeraldine molina">

strong

Muestra el texto en negrita. Ejemplo:

Esto es un ejemplo Esto es un ejemplo

time

Define una fecha o hora específica. Ejemplo:

La reunios esta programada para el 10 de junio

details: Crea un elemento desplegable para mostrar u ocultar contenido adicional.
summary: Define el texto visible para el elemento.

figure

<img src="img/img.jpg">

code y pre

code: Se utiliza para representar fragmentos de código en línea dentro de un documento. Se utiliza para resaltar y distinguir visualmente el código del texto normal.
pre: se utiliza para mostrar texto preformateado, es decir, texto en el que se mantiene la estructura de espacios en blanco, saltos de línea y otros caracteres tal como están en el código fuente.

em, del y mark

em:Muestra el texto en cursiva o énfasis.
del:Muestra el texto como eliminado o tachado.
Mark:Resalta o marca una parte del texto.
Ejemplo:

La historia de la programación se encuentra fuertemente relacionada con la computación, puesto que a partir de la aparición de los ordenadores fue que esta empezó a tener relevancia para la sociedad, debido a la velocidad con la cual se podían ejecutar tareas mecánicas mediante la codificación de comandos.

source: especifica recursos de medios múltiples para los elementos picture, audio, video.

table, tr, th y td

table: encierra todas las filas y columnas de la tabla.
tr: esta etiqueta definen cada fila de la tabla y encierran todas las columnas.
th: representa a una celda de encabezado en una tabla (table). Las celdas de encabezado están pensadas para proveer información de encabezado para las celdas de datos td.
td: se utiliza para definir una celda de datos en una tabla. Ejemplo:


Libro Año
Regeneration 1991
Sálvese quien pueda 1996

nav, ul, li

nav: define una sección de navegación dentro de un documento o un área de navegación global.

ul: es una herramienta esencial para crear listas desordenadas en un documento web. Se utiliza para presentar una serie de elementos relacionados que no tienen un orden específico.

li: es una etiqueta que se utiliza para representar un elemento de lista en una lista ordenada.
Ejemplo:

  • Sobre nosotros
